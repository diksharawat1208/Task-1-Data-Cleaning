Task 1 - Data Cleaning and Preprocessing
Objective

The objective of this task was to clean and preprocess a raw dataset using Microsoft Excel to improve data quality and consistency.

Dataset

Marketing Campaign Dataset

Tools Used
Microsoft Excel
Data Cleaning Steps Performed
Checked the dataset for missing values.
Handled missing values appropriately.
Checked for duplicate records and removed them if found.
Renamed column headers to lowercase with underscores.
Standardized text values by removing extra spaces.
Converted the customer registration date to a consistent DD-MM-YYYY format.
Verified data types for numeric and date columns.
Saved the cleaned dataset.
